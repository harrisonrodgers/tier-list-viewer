# Tier List Viewer

Create custom tier lists by arranging images in folders and controlling order with filenames.

## Description

Tier List Viewer is a simple tool that allows you to view tier lists. Create folders for your tiers, place your images inside, and order by setting the folder/file name. View the tier list in your web browser by running the `run.py` with `Terminal.app` or `python.exe`.

## Features

* Folders of images used for tiers.
* Image names used for ranking within tiers.
* Single dependency on python.

## Get Started

1. Download and unzip this repository.
2. Customize:
    * Rename the `Example` folder to set the name of your tier list.
    * Place your images in the tier folders.
    * Inside each tier folder, rename the images to sort them.
3. Right click on `run.py` and set always 'Open With' to `Terminal.app` or `python.exe`.

## Advanced

* **Rename tier names:** edit the `000.svg` with a text editor to change the text inside.
* **Custom tier images:** delete the `000.svg` and add your own images with name `000`.
